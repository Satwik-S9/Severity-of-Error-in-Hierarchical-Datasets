## Dependencies
1. python 3.7+
2. pytorch 1.10+
3. pandas
4. networkx
5. pyyaml
6. pickle
7. numpy
8. barbar

## How to run the code 
1. ```python main.py```
2. To see options run ```python main.py --help```
3. The `src/` folder considers all the scripts for models and datasets that needs to be changed for a custom run
